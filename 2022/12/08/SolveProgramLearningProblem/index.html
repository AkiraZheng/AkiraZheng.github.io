    • git config --global user.email "дҪ зҡ„йӮ®з®ұ"

  • е…ӢйҡҶйЎ№зӣ®

    • git clone йЎ№зӣ®ең°еқҖ

2.2.3 дёҫдҫӢжӣҙж–°йЎ№зӣ®еҲ°иҝңзЁӢд»“еә“

дёүгҖҒдә‘жңҚеҠЎеҷЁиҝңзЁӢиҝһжҺҘ

1 йҳҝйҮҢдә‘

1.1 windowsзі»з»ҹ

  • еҲӣе»әз”ЁдәҺйғЁзҪІзҡ„з«ҜеҸЈеҸ·

    • иҝӣе…ҘйҳҝйҮҢдә‘жңҚеҠЎеҷЁжҺ§еҲ¶еҸ°йҖүжӢ©еҜ№еә”зҡ„жңҚеҠЎеҷЁзӮ№еҮ»иҝӣе…Ҙ

    • зӮ№еҮ»дёҠж–№Tabзҡ„е®үе…Ёз»„->з®ЎзҗҶи§„еҲҷ->жүӢеҠЁж·»еҠ 

  • дҝ®ж”№зі»з»ҹзҷ»еҪ•еҜҶз Ғ

    • иҝӣе…ҘйҳҝйҮҢдә‘жңҚеҠЎеҷЁжҺ§еҲ¶еҸ°йҖүжӢ©еҜ№еә”зҡ„жңҚеҠЎеҷЁзӮ№еҮ»иҝӣе…Ҙ

    • зӮ№еҮ»дёҠж–№Tabзҡ„е®һдҫӢиҜҰжғ…->еҹәжң¬дҝЎжҒҜ->йҮҚзҪ®еҜҶз Ғ

    • жіЁж„ҸпјҡдёҚиғҪдҝ®ж”№зҷ»еҪ•еҗҚadministratorпјҢеҸӘиғҪдҝ®ж”№еҜҶз ҒпјҢеҗҰеҲҷдјҡдҝ®ж”№еӨұиҙҘ

  • иҝңзЁӢзҷ»еҪ•

    • еңЁиҮӘе·ұзҡ„з”өи„‘дё»жңәдёӯзӮ№еҮ»ејҖе§Ӣ->иҝҗиЎҢ->иҫ“е…Ҙmstsc->зЎ®е®ҡ->иҫ“е…ҘжңҚеҠЎеҷЁзҡ„е…¬зҪ‘IPең°еқҖ->иҝһжҺҘ->иҫ“е…Ҙз”ЁжҲ·еҗҚadministrator->иҫ“е…ҘеҜҶз Ғ->зЎ®е®ҡ

    • жҲ–иҖ…еңЁз”өи„‘зҡ„жҗңзҙўж ҸдёӯжҗңзҙўиҝңзЁӢжЎҢйқўиҝһжҺҘпјҢз„¶еҗҺиҫ“е…ҘжңҚеҠЎеҷЁзҡ„е…¬зҪ‘IPең°еқҖ->иҝһжҺҘ->иҫ“е…Ҙз”ЁжҲ·еҗҚadministrator->иҫ“е…ҘеҜҶз Ғ->зЎ®е®ҡ

  • иҝңзЁӢж–Үд»¶дј иҫ“

    • йҳҝйҮҢдә‘жҸҗдҫӣзҡ„дёҠдј ж–Үд»¶еҠҹиғҪ
      • 1.еңЁиҝңзЁӢжЎҢйқўиҝһжҺҘеүҚзӮ№еҮ»жҳҫзӨәйҖүйЎ№->еёёи§„Tabдёӯжң¬ең°иө„жәҗ
      • 2.зӣҙжҺҘеңЁжң¬жңәдёӯйҖҡиҝҮеӨҚеҲ¶зІҳиҙҙдј йҖ’ж–Үд»¶

еӣӣгҖҒдҪҝз”ЁVisual StudioдёҺGithubиҝӣиЎҢд»Јз Ғз®ЎзҗҶеҮәзҺ°зҡ„й—®йўҳ

1.1 зӘҒз„¶еҮәзҺ°вҖңжңӘиғҪжҺЁйҖҒеҲ°иҝңзЁӢд»“еә“вҖқзҡ„й—®йўҳ

  • й—®йўҳеҰӮдёӢпјҡ

  • и§ЈеҶіж–№жЎҲ

    • 1пјүйҰ–е…Ҳе…ҲдҪҝз”Ёйӯ”жі•з§‘еӯҰдёҠзҪ‘
    • 2пјүжҺҘзқҖеңЁcmdдёӯжөӢиҜ•иғҪдёҚиғҪpingйҖҡgithub.comгҖӮping github.comпјҢй’ҲеҜ№pingдёҚйҖҡзҡ„жғ…еҶөпјҢйңҖиҰҒдҝ®ж”№hostsж–Үд»¶гҖӮ
    • 3пјүжү“ејҖC:\Windows\System32\drivers\etcзӣ®еҪ•дёӢзҡ„hostsж–Үд»¶

    • 4пјүеңЁhostsж–Үд»¶дёӯж·»еҠ еҰӮдёӢеҶ…е®№е№¶дҝқеӯҳ markdown 192.30.255.112 github.com git 185.31.16.184 github.global.ssl.fastly.net
    • 5пјүеҶҚж¬ЎеңЁcmdдёӯжөӢиҜ•иғҪдёҚиғҪpingйҖҡgithub.comгҖӮping github.comпјҢеҰӮжһңиғҪpingйҖҡпјҢе°ұеҸҜд»ҘеңЁvscodeдёӯиҝӣиЎҢpushж“ҚдҪңдәҶгҖӮ

1.2 еҸідҫ§е·Ҙе…·ж Ҹж¶ҲеӨұ

  • и§ЈеҶіж–№жЎҲ
