1.1 еҚҒиҝӣеҲ¶ -> дәҢиҝӣеҲ¶

еңЁеҒҡз¬”иҜ•ж—¶пјҢжңҖеҘҪиҝҳжҳҜз”ЁзҺ°жңүзҡ„иҪ¬жҚўеҮҪж•°пјҢж–№дҫҝзӮ№пјҡ

  • bitset<еӨҡе°‘дҪҚзҡ„дәҢиҝӣеҲ¶> b(decNum);

жңҖеҗҺжҳҜжңҖеҺҹе§Ӣзҡ„ж–№жі•пјҢзәҜC++е®һзҺ°пјҡ

  • stoi(binNum, жҢҮй’Ҳ, иҝӣеҲ¶еҹәж•°);

жңҖеҺҹе§Ӣзҡ„ж–№жі•пјҡ

е…¶е®һеҸҜд»ҘдёҚйңҖиҰҒеҮҪж•°пјҢзӣҙжҺҘйҖҡиҝҮ'a' - 'A'зҡ„е·®еҖјиҝӣиЎҢиҪ¬жҚўпјҢз„¶еҗҺйҖҗдёӘиҪ¬жҚўе°ұиЎҢ - е°ҸеҶҷеӯ—жҜҚиҪ¬еӨ§еҶҷеӯ—жҜҚпјҡchar c = c0 - ('a' - 'A'); - еӨ§еҶҷеӯ—жҜҚиҪ¬е°ҸеҶҷеӯ—жҜҚпјҡchar c = c0 + ('a' - 'A');
